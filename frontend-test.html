Interface de Test - Microservices

Testez tous vos endpoints User, Product et Order en un clic

User Service (8082)
POST /api/auth/register S'inscrire
POST /api/auth/login Se connecter
GET /api/users Tous les utilisateurs (Admin)

Product Service (8081)

POST /api/products Créer produit (Admin)
PUT /api/products/{id} Modifier produit (Admin)

Order Service (8083)

PUT /api/orders/{id}/status Modifier statut
PUT /api/orders/{id}/cancel Annuler commande
GET /api/orders/stats Statistiques
